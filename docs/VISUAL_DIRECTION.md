# Visual Direction

YOZI Dao Runtime should feel like Song dynasty minimalism meeting a code-based metaphysics interface.

The homepage should feel poetic, mysterious, restrained, warm, and premium.

## Core Idea

The main visual is a living symbolic organism made entirely from code characters, Bagua symbols, simple lines, dots, coordinates, and subtle motion.

It should suggest a digital cultivation system without using literal religious, fantasy, landscape, or character illustration.

## Use

- Canvas or SVG.
- Text symbols.
- Bagua trigrams.
- Thin low-opacity lines.
- Sparse dots and coordinate-like marks.
- Warm paper-like backgrounds.
- Restrained ink-like foreground colors.
- Slow, subtle, organic motion.
- Generous negative space.

## Avoid

- Cyberpunk neon.
- Generic Matrix rain.
- Bright green terminal aesthetics.
- Heavy blur, glow, or noisy particles.
- Mountains, temples, robes, statues, concrete Taoist imagery.
- Anime style.
- Cute cartoon style.
- Stock imagery.
- Dense decorative UI cards.
- Visuals that compete with the title.

## Palette

Current palette direction:

- Warm paper base: `#f6f3eb`
- Muted section base: `#ebe5d8`
- Ink text: `#15130f`
- Warm secondary text: `#3a352c`
- Muted interface marks: `#746a55`, `#514a3b`

The palette should remain warm and quiet, but avoid becoming flat beige. Small tonal contrast is preferred over saturated color.

## Typography

The hero title uses a serif stack for a literary, formal tone. Interface labels use small uppercase tracking for a code-system feeling.

Avoid scaling body text with viewport width. Keep letter spacing non-negative.

## Animation Tone

Motion should be almost meditative:

- Growth should feel slow and intentional.
- Breathing should be barely perceptible.
- Pointer response should feel like attention, not a game effect.
- Scroll dispersal should feel like symbols returning to field state.

If an effect calls attention to itself, reduce it.

## Code Hermit Tone

The Code Hermit is a small observer, not a mascot.

It may be relaxed and slightly humorous, but should remain quiet:

- No detailed face.
- No costume.
- No fantasy silhouette.
- No religious figure treatment.
- No exaggerated cuteness.

Think: a tiny system operator sitting beneath the root, quietly watching the branch runtime breathe.
